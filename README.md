# dbrownell_ClaudePluginTest

Some sample code for a Claude plugin that define simple skills. This code was generated by Claude.

## To Test Locally

1. `claude --plugin-dir ./` (when run from the repositor  root)
2. In claude...
3. `/vowel-upper claude is fun!` -> `"clAUdE Is fUn!"`
4. `/math-tens add 10 2` ->

```
Operation: add 10 2
Raw result: 12
Rounded to nearest 10: 10
```

## To Distribute

1. In claude...
2. `/plugin marketplace add davidbrownell/ClaudePluginTest`
3. `/plugin install string-math-tools@davidbrownell-ClaudePluginTest`
4. `/string-math-tools:vowel-upper claude is fun!` -> `"clAUdE Is fUn!"`
5. `/string-math-tools:math-tens add 10 2` ->

```
Operation: add 10 2
Raw result: 12
Rounded to nearest 10: 10
```
